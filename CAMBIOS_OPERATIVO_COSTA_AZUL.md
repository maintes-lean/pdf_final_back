# Cambios operativos Costa Azul

Se agregaron módulos internos para profesionalizar el uso diario de la agencia:

- Dashboard operativo.
- Búsqueda global.
- Tareas simples por cliente/viaje/cotización.
- Checklist de viaje con ítems automáticos.
- Timeline histórico del cliente.
- Registro manual de aprobación de cotización.
- Enlace rápido a WhatsApp Web.

Las tablas se crean automáticamente al usar `/api/operational/*`.
También se incluye `scripts/sql/operational_tools.sql` como respaldo manual.
